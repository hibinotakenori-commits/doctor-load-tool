# 医師負荷可視化ツール

病院の診療科で新規入院の割り振り判断を支援するツールです。  
医師個人の評価が目的ではなく、**チーム支援・医療安全**のために使用します。

## セットアップ

```bash
# 依存パッケージをインストール
pip install streamlit pandas

# アプリを起動
cd doctor_load_tool
streamlit run app.py
```

## 機能一覧

| 画面 | 内容 |
|------|------|
| ダッシュボード | 本日の医師一覧・負荷スコアの確認 |
| 日次入力 | 各医師の当日状況を入力・保存 |
| アサイン支援 | 新規入院の割り振り候補を自動表示 |

## 負荷スコアの計算式

| 項目 | 点数 |
|------|------|
| 受け持ち患者数 | × 3点 |
| 重症患者数 | × 10点 |
| 新規入院数 | × 12点 |
| 退院予定数 | × −4点（負荷軽減） |
| 外来あり | +15点 |
| 当直明け | +20点 |
| 主観的余裕 1〜5 | 余裕1→+25点、余裕5→+5点（逆転加点） |

## 負荷レベル

| レベル | スコア |
|--------|--------|
| 🟢 低負荷 | 40点未満 |
| 🟡 中等度 | 40〜69点 |
| 🔴 高負荷 | 70点以上 |

## データ保存

- 入力データは `data/doctors.csv` に自動保存されます
- CSVが存在しない場合、サンプルデータで起動します
- 同じ日付・医師名のデータは上書きされます
